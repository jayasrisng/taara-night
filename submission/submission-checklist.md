# Submission Checklist

## Official Devpost Requirements Checked

Source: https://redditgameswithahook.devpost.com/

- Deadline: July 15, 2026 at 6:00 PM PDT.
- Build requirement: a game on Devvit Web using Interactive Posts.
- Compliance requirement: Devvit Rules.
- Submission requirement: app listing link on `developer.reddit.com`.
- Submission requirement: demo post link to the subreddit and a public post running the game.
- Judging note: judges primarily evaluate community play from the demo link, so the public post must be self-explanatory.
- Optional: complete the Developer Platform feedback survey for Best Feedback prize eligibility.
- Eligibility summary shown on Devpost: above legal age of majority in country of residence; all countries/territories except Brazil, Crimea, Cuba, Iran, North Korea, Quebec, and Russia.

## Judging Criteria

- Delightful UX: exciting layout/theme; easy and fun to uncover the app.
- Polish: close to publishable, Devvit Rules compliant, understandable through example posts.
- Reddit-y: community-minded, fresh identity, brings a community together.
- Hook-y: reason to return regularly through anticipation, goals, evolving experiences, or fresh opportunities.
- Phaser Innovation: creative/effective use of Phaser that elevates gameplay, polish, and technical execution.

## Before Devpost

- Run all local gates from `taara-connect/`.
- Playtest on `r/taara_connect_dev`.
- Publish or upload the final app version.
- Create the public play/demo post in `r/TaaraNight`.
- Open the public post in Reddit and solve one puzzle end to end.
- Confirm Results, Comment/Copy share, My Sky, and ambient audio all work in the Reddit webview.
- Upload `submission/taaranight-demo.mp4` to Devpost if Devpost asks for a video.
- Paste `submission/devpost-submission.md` into Devpost.
- Add the app listing link: `https://developers.reddit.com/apps/taara-connect`.
- Add the public `r/TaaraNight` demo post link.
- Add the pushed GitHub source link.
- Complete the optional Developer Platform feedback survey for the Best Feedback prize.
