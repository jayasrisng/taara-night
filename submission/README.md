# TaaraNight Submission Package

This folder contains the materials to submit TaaraNight to Reddit's Games with a Hook Hackathon.

## Files

- `taaranight-demo.mp4` — final demo video, 55.47s, 1080x1920, H.264/AAC.
- `devpost-submission.md` — text to paste into Devpost.
- `submission-checklist.md` — final pre-submit checklist.
- `test-plan.md` — exact verification steps for Reddit playtest and local checks.
- `push-and-submit.md` — exact commands and submission flow.

## Primary Links

- Public community: https://www.reddit.com/r/TaaraNight/
- Dev/test community: https://www.reddit.com/r/taara_connect_dev/
- Devvit app slug: `taara-connect`

Use `r/TaaraNight` for the public submission/demo post. Use `r/taara_connect_dev` only for final private playtest.
